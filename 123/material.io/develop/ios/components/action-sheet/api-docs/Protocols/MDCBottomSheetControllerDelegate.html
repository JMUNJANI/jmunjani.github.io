MDCBottomSheetControllerDelegate

@protocol MDCBottomSheetControllerDelegate <NSObject>

Delegate for MDCBottomSheetController.

  • -bottomSheetControllerDidDismissBottomSheet:

    Called when the user taps the dimmed background or swipes the bottom sheet off to dismiss the bottom sheet. Also called with accessibility escape two finger Z gestures.

    This method is not called if the bottom sheet is dismissed programatically.

    Declaration

    Objective-C

    - (void)bottomSheetControllerDidDismissBottomSheet:
    (nonnull MDCBottomSheetController *)controller;

    Parameters

    controller

    The MDCBottomSheetController that was dismissed.
  • -bottomSheetControllerStateChanged:state:

    Called when the state of the bottom sheet changes.

    Note: See what states the sheet can transition to by looking at MDCSheetState.

    Declaration

    Objective-C

    - (void)bottomSheetControllerStateChanged:
            (nonnull MDCBottomSheetController *)controller
                                    state:(MDCSheetState)state;

    Parameters

    controller

    The MDCBottomSheetController that its state changed.
    state

    The state the sheet changed to.
  • -bottomSheetControllerDidChangeYOffset:yOffset:

    Called when the Y offset of the sheet’s changes in relation to the top of the screen.

    Declaration

    Objective-C

    - (void)bottomSheetControllerDidChangeYOffset:
            (nonnull MDCBottomSheetController *)controller
                                      yOffset:(CGFloat)yOffset;

    Parameters

    controller

    The MDCBottomSheetController that its Y offset changed.
    yOffset

    The Y offset the bottom sheet changed to.
