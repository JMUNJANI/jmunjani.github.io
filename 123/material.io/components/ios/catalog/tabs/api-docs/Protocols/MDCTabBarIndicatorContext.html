MDCTabBarIndicatorContext

@protocol MDCTabBarIndicatorContext <NSObject>

Information about the context in which a tab bar indicator will be displayed.

  • item

    The tab bar item for the indicated tab.

    Declaration

    Objective-C

    @property (readonly, nonatomic, nonnull) UITabBarItem *item;
  • bounds

    The full bounds of the tab’s view.

    Any paths should be created relative to this coordinate space.

    Declaration

    Objective-C

    @property (readonly, nonatomic) CGRect bounds;
  • contentFrame

    The frame for the tab’s primary content in its coordinate space.

    For title-only tabs, this is the frame of the title text. For image-only tabs, this is the frame of the primary image. For title-and-image tabs, it is the union of the title and primary image’s frames.

    Declaration

    Objective-C

    @property (readonly, nonatomic) CGRect contentFrame;
