MDCActivityIndicatorTransition

@interface MDCActivityIndicatorTransition : NSObject

Describes an animation that can be provided to an MDCActivityIndicator instance to perform before or after its standard cycle animation.

  • animation

    The animations to be performed by MDCActivityIndicator. In this block add CAAnimations to be animated before or after MDCActivityIndicator’s cycle animation. MDCActivityIndicator will trigger these animations and call completion after they complete.

    Declaration

    Objective-C

    @property (readwrite, copy, nonatomic, nonnull)
    MDCActivityIndicatorAnimation animation;
  • completion

    The completion block to call after animation’s completion. This should be used to clean up any layers placed and animating on the MDCActivityIndicator.

    Declaration

    Objective-C

    @property (readwrite, copy, nonatomic, nullable) void (^)(void) completion;

    The duration of the animation.
