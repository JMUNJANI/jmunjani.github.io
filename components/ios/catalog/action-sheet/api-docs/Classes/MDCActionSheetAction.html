MDCActionSheetAction

@interface MDCActionSheetAction
    : NSObject <NSCopying, UIAccessibilityIdentification>

An instance of MDCActionSheetAction is passed to MDCActionSheetController to add an action to the action sheet.

  • +actionWithTitle:image:handler:

    Returns an action sheet action with the populated given values.

    Declaration

    Objective-C

    + (nonnull instancetype)actionWithTitle:(nonnull NSString *)title
                                  image:(nullable UIImage *)image
                                handler:
                                    (MDCActionSheetHandler _Nullable)handler;

    Parameters

    title

    The title of the list item shown in the list
    image

    The icon of the list item shown in the list
    handler

    A block to execute when the user selects the action.

    Return Value

    An initialized MDCActionSheetAction object.

  • -init

    Action sheet actions must be created with actionWithTitle:image:handler:

    Declaration

    Objective-C

    - (nonnull instancetype)init;
  • title

    Title of the list item shown on the action sheet.

    Action sheet actions must have a title that will be set within actionWithTitle:image:handler: method.

    Declaration

    Objective-C

    @property (readonly, nonatomic, nonnull) NSString *title;
  • image

    Image of the list item shown on the action sheet.

    Action sheet actions must have an image that will be set within actionWithTitle:image:handler: method.
